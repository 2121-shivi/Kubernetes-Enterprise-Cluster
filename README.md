# Kubernetes Enterprise Cluster

A 5-node Kubernetes cluster deployed on VMware Cloud Director.

## Project Overview

This project demonstrates an enterprise Kubernetes environment using:

- Kubernetes Cluster
- VMware Cloud Director
- Nginx
- Apache
- MySQL
- PostgreSQL
- Redis
- MinIO
- Persistent Volumes
- ConfigMaps
- Metrics Server
- Kubernetes Dashboard

---

## Cluster Architecture

| Node | Role |
|------|------|
| k8s-master | Control Plane |
| worker-1 | Web Services |
| worker-2 | Database Services |
| worker-3 | Storage Services |
| worker-4 | Application Services |
