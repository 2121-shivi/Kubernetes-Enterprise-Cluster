# Kubernetes Enterprise Cluster Setup Guide

## Environment

| Component | Value |
|----------|--------|
| Platform | VMware Cloud Director |
| OS | Ubuntu 22.04 LTS |
| Container Runtime | containerd |
| Kubernetes | kubeadm |
| Cluster | 1 Master + 4 Workers |

---

# Cluster Architecture

| Node | Role |
|------|------|
| k8s-master | Control Plane |
| worker-1 | Web Server |
| worker-2 | Database |
| worker-3 | Cache |
| worker-4 | Storage |

---

# Step 1: Update All Nodes

Run on all VMs.

```bash
apt update
apt upgrade -y
```

---

# Step 2: Disable Swap

```bash
swapoff -a
sed -i '/ swap / s/^/#/' /etc/fstab
```

Verify:

```bash
free -h
```

---

# Step 3: Enable Kernel Modules

```bash
modprobe overlay
modprobe br_netfilter
```

```bash
cat <<EOF | tee /etc/modules-load.d/k8s.conf
overlay
br_netfilter
EOF
```

---

# Step 4: Configure Networking

```bash
cat <<EOF | tee /etc/sysctl.d/k8s.conf
net.bridge.bridge-nf-call-iptables=1
net.bridge.bridge-nf-call-ip6tables=1
net.ipv4.ip_forward=1
EOF
```

Apply:

```bash
sysctl --system
```

---

# Step 5: Install Containerd

```bash
apt install containerd -y
```

```bash
mkdir -p /etc/containerd

containerd config default > /etc/containerd/config.toml
```

```bash
systemctl restart containerd
systemctl enable containerd
```

---

# Step 6: Install Kubernetes Packages

```bash
apt update
apt install apt-transport-https ca-certificates curl gpg -y
```

```bash
mkdir -p /etc/apt/keyrings

curl -fsSL https://pkgs.k8s.io/core:/stable:/v1.30/deb/Release.key \
| gpg --dearmor -o /etc/apt/keyrings/kubernetes-apt-keyring.gpg
```

```bash
echo 'deb [signed-by=/etc/apt/keyrings/kubernetes-apt-keyring.gpg] \
https://pkgs.k8s.io/core:/stable:/v1.30/deb/ /' \
> /etc/apt/sources.list.d/kubernetes.list
```

```bash
apt update

apt install kubelet kubeadm kubectl -y

apt-mark hold kubelet kubeadm kubectl
```

---

# Step 7: Initialize Master Node

Run only on k8s-master.

```bash
kubeadm init --pod-network-cidr=192.168.0.0/16
```

Configure kubectl:

```bash
mkdir -p $HOME/.kube

cp /etc/kubernetes/admin.conf $HOME/.kube/config
```

---

# Step 8: Install Calico

```bash
kubectl apply -f https://raw.githubusercontent.com/projectcalico/calico/v3.28.0/manifests/calico.yaml
```

Verify:

```bash
kubectl get pods -n kube-system
```

---

# Step 9: Join Worker Nodes

Run the join command generated by kubeadm init.

```bash
kubeadm join MASTER-IP:6443 --token TOKEN \
--discovery-token-ca-cert-hash sha256:HASH
```

Verify:

```bash
kubectl get nodes
```

---

# Step 10: Label Nodes

```bash
kubectl label node worker-1 role=web

kubectl label node worker-2 role=database

kubectl label node worker-3 role=cache

kubectl label node worker-4 role=storage
```

---

# Step 11: Create Namespaces

```bash
kubectl create namespace web

kubectl create namespace database

kubectl create namespace cache

kubectl create namespace storage
```

---

# Step 12: Deploy Applications

## Nginx

```bash
kubectl apply -f manifests/nginx-deployment.yaml

kubectl apply -f manifests/nginx-service.yaml
```

## Apache

```bash
kubectl apply -f manifests/apache-deployment.yaml
```

## MySQL

```bash
kubectl apply -f manifests/mysql-pvc.yaml

kubectl apply -f manifests/mysql-deployment.yaml
```

## PostgreSQL

```bash
kubectl apply -f manifests/postgres-deployment.yaml
```

## Redis

```bash
kubectl apply -f manifests/redis-deployment.yaml
```

## MinIO

```bash
kubectl apply -f manifests/minio-deployment.yaml

kubectl apply -f manifests/minio-service.yaml
```

---

# Step 13: Scaling

```bash
kubectl scale deployment nginx --replicas=5 -n web
```

Verify:

```bash
kubectl get pods -n web
```

---

# Step 14: Self-Healing

Delete a pod:

```bash
kubectl delete pod POD_NAME -n web
```

Watch recreation:

```bash
kubectl get pods -n web -w
```

---

# Step 15: Rolling Updates

```bash
kubectl set image deployment/nginx nginx=nginx:1.27 -n web
```

Check rollout:

```bash
kubectl rollout status deployment nginx -n web
```

Rollback:

```bash
kubectl rollout undo deployment nginx -n web
```

---

# Step 16: Persistent Storage

```bash
kubectl get pv

kubectl get pvc -n database
```

---

# Step 17: ConfigMap

```bash
kubectl create configmap website-content \
--from-file=index.html \
-n web
```

Restart deployment:

```bash
kubectl rollout restart deployment nginx -n web
```
